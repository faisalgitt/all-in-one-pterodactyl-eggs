# All in One Pterodactyl Eggs 🥚

A complete collection of fully working generic eggs for the **Pterodactyl Panel**. This repository contains pre-configured eggs for all major runtimes – everything you need to deploy any application in one place.

[![Discord](https://img.shields.io/badge/Discord-Join%20Server-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://dsc.gg/cyerox)
[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/KopeeDev/all-in-one-pterodactyl-eggs)

## 📦 Included Eggs

| Language/Runtime | Description |
|------------------|-------------|
| **Bun** | Incredibly fast JavaScript runtime, transpiler, and bundler |
| **Dart** | Generic Dart CLI egg with pubspec.yaml support |
| **Deno** | Secure runtime for JavaScript and TypeScript |
| **Elixir** | Functional, concurrent language running on BEAM |
| **C# (.NET)** | Generic C# egg for .NET projects |
| **Java** | Universal Java runtime container |
| **Go** | Golang egg with compilation support |
| **Luvit** | Lua runtime with Node.js-style APIs |
| **Node.js** | Generic Node.js egg with npm support |
| **Nodemon** | Node.js with auto-restart on file changes |
| **Python** | Generic Python egg with pip support |
| **Rust** | Rust runtime with Cargo support |

## ✨ Features

- ✅ **100% Working** – All eggs are tested and production-ready
- ✅ **Easy Import** – Simple JSON format, compatible with Pterodactyl v2
- ✅ **Git Support** – Automatic cloning from repositories
- ✅ **User Upload** – Option to manually upload files
- ✅ **Dependency Management** – Automatic installation of packages
- ✅ **Customizable** – Easily modify variables for your needs

## 🚀 Quick Start

1. **Download** the egg JSON file for your desired runtime
2. **Import** to your Pterodactyl panel (Nests → Import Egg)
3. **Configure** variables (Git repo, startup command, etc.)
4. **Create** a new server using the imported egg
5. **Deploy** your application!

## 📋 Requirements

- Pterodactyl Panel **v1.0+** (v2 compatible)
- Wings **v1.0+**

## 🔧 Usage Examples

### Node.js Application
```bash
# Startup command
node index.js

# Variables
Git Repository: https://github.com/user/bot.git
Main File: index.js
```

### Python Bot
```bash
# Startup command
python main.py

# Variables
Git Repository: https://github.com/user/python-bot.git
Requirements: requirements.txt
```

### Java Server
```bash
# Startup command
java -Xms128M -Xmx${SERVER_MEMORY}M -jar server.jar

# Variables
Git Repository: https://github.com/user/minecraft-server.git
Jar File: server.jar
```
